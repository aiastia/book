【题材锁定——最高优先级，违反即失败】
1. 所有内容（角色身份/背景/能力、世界观设定、情节场景）必须严格属于「{genre}」题材，不得跨题材。
2. 严禁现代科技/科幻元素乱入到非科技题材中：如果{genre}属于古典/架空/奇幻/玄幻/仙侠/武侠/历史/言情/宫斗/种田 等非科技背景，绝对禁止出现 科学家、机甲、机器人、人工智能、基因工程、赛博、星际、实验、芯片、实验室、机械、引擎、能源、手机、电脑、网络 等任何现代科技/科幻元素。
3. 力量体系必须与原作同类（原作修仙→新书修仙；原作异能→新书异能；原作魔法→新书魔法；原作武学→新书武学），不得自行升级为高科技或混搭科技。
4. 反之，如果{genre}本身是科技/科幻/都市/末世类，则按其体系设计，不要硬塞修仙/魔法等超自然体系。
5. 当不确定某设定是否符合题材时，一律向原作的题材气质靠拢，绝不自行升级为高科技设定。
